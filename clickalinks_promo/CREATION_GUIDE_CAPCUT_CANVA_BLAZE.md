### Step 2: Create Voiceover with CapCut Text-to-Speech

**‚≠ê RECOMMENDED: Use CapCut's Built-in Text-to-Speech**

**Why Use Text-to-Speech:**
- ‚úÖ No microphone needed
- ‚úÖ Perfect pronunciation
- ‚úÖ Professional quality
- ‚úÖ Multiple voice options
- ‚úÖ Easy to edit and re-generate
- ‚úÖ Perfect for non-native English speakers

**How to Use CapCut Text-to-Speech:**

1. **Prepare Your Script:**
   - Open script from `01_scripts/` folder
   - Copy the text
   - Remove timestamps like `[0:00-0:05]`
   - Remove stage directions like `[Hook]`, `[Problem Setup]`
   - Keep only the spoken text
   - Add punctuation for natural pauses

2. **In CapCut:**
   - Tap/Click "Text" tool
   - Add text box
   - Paste your prepared script
   - Select the text
   - Look for "Text-to-Speech" or "Voice" icon/option
   - Tap/Click to convert text to speech

3. **Choose Voice:**
   - Select voice type (male/female, energetic/professional)
   - Choose accent (British/American if available)
   - Adjust speed if needed (usually 1.0x is best)
   - Generate speech

4. **Review & Adjust:**
   - Listen to generated voiceover
   - If not satisfied, try different voice
   - Adjust speed/pitch if needed
   - Re-generate until perfect

**Tips for Best Results:**
- Add punctuation for natural pauses (commas, periods, ellipsis)
- Break long sentences into shorter ones
- Use ALL CAPS sparingly for emphasis
- Test 2-3 different voices, pick the best
- See detailed guide: `CAPCUT_TEXT_TO_SPEECH_GUIDE.md`

**Alternative: Record Your Own Voice (If Preferred)**

**Option A: Record on Phone**
1. Open voice recorder app
2. Read script naturally (practice 2-3 times first)
3. Record in quiet space
4. Save as: `voiceover_[name].mp3` or `.m4a`

### Step 5: Create Video in CapCut

#### A. Start New Project

**Mobile (CapCut App):**
1. Open CapCut app
2. Tap "+ New Project"
3. Select aspect ratio:
   - **TikTok/Reels:** 9:16 (vertical)
   - **YouTube/Facebook:** 16:9 (horizontal)
   - **Instagram Square:** 1:1

**Desktop (CapCut PC):**
1. Open CapCut
2. Click "New Project"
3. Set resolution: 1080p (1920x1080 or 1080x1920)
4. Frame rate: 30fps

#### B. Import Media

1. Tap/Click "Import" or "+"
2. Add files:
   - Voiceover audio
   - Screen recordings
   - Background music
   - Any images/graphics
3. Drag to timeline in order

#### C. Edit Timeline

**Arrange Clips:**
- Voiceover on main audio track
- Screen recordings on video track
- Background music on separate audio track (lower volume)

**Trim Clips:**
- Cut out mistakes
- Remove dead air
- Keep best takes only

**Adjust Speed:**
- Speed up boring parts (2x-3x)
- Slow down important moments (0.5x)
- Keep voiceover at normal speed

#### D. Add Text Overlays

**Using Your Style Guide:**
1. Tap/Click "Text" tool
2. Add text from your script
3. Apply settings from `video_style_guide.txt`:
   - **Font:** Montserrat Bold or similar
   - **Size:** 60px for headlines (adjust for platform)
   - **Color:** White (on dark) or #1e293b (on light)
   - **Position:** Center, safe zone

**Key Text Overlays to Add:**
- Hook text (first 3 seconds)
- Key phrases from script
- Stats: "¬£1 Per Day", "2,000 Spots"
- CTA: "Visit clickalinks.com"
- "Link in bio" (for TikTok/Instagram)

**Text Animations:**
- Fade In (0.3-0.5 seconds)
- Slide Up (0.4-0.6 seconds)
- Scale (for emphasis)
- Match timing to voiceover

#### E. Apply Brand Colors

**For Text:**
1. Select text
2. Tap "Style" or "Color"
3. Use brand colors:
   - Orange: #ff6b35
   - Cyan: #06b6d4
   - Blue: #3b82f6
   - Green: #10b981
   - Pink: #ec4899
   - Yellow: #fbbf24

**For Backgrounds/Overlays:**
1. Add shape overlay
2. Apply gradient using brand colors
3. Adjust opacity (20-40% for subtle)

#### F. Add Transitions

**Between Clips:**
- Fade (0.3-0.5s) - Professional
- Slide (0.4-0.6s) - Dynamic
- Zoom (0.3-0.5s) - Energetic

**Tips:**
- Don't overuse transitions
- Match transition to video pace
- Keep it smooth

#### G. Adjust Audio

**Voiceover:**
- Volume: 100% (foreground)
- Clear, crisp
- Remove background noise if needed

**Background Music:**
- Volume: 20-30% (background)
- Fade in/out: 1-2 seconds
- Lower during voiceover

**Balance:**
- Voiceover should be clear
- Music should support, not overpower

#### H. Add Effects (Optional)

**Screen Recording Effects:**
- Zoom/Pan for movement
- Highlight boxes for important areas
- Blur sensitive information (payment details)

**Visual Effects:**
- Subtle glow on important text
- Particle effects (sparingly)
- Color correction (enhance brand colors)

#### I. Final Review

**Checklist:**
- [ ] All text is readable
- [ ] Brand colors used correctly
- [ ] Audio levels balanced
- [ ] No spelling errors
- [ ] CTA is clear
- [ ] Website URL visible
- [ ] Duration matches script
- [ ] Quality is high (1080p)

#### J. Export Video

**Settings:**
- **Resolution:** 1080p (1920x1080 or 1080x1920)
- **Frame Rate:** 30fps
- **Format:** MP4
- **Quality:** High

**Naming:**
- Save as: `hero_launch_facebook_1080p.mp4`
- Store in: `clickalinks_promo/04_final_exports/videos_1080p/`

1. Click on any element
2. Click color picker
