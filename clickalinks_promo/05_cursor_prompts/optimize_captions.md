# Cursor Prompts for Optimizing Social Media Captions

Use these prompts to optimize and improve your social media captions for better engagement.

---

## Prompt 1: Optimize Caption for Engagement

```
Optimize this social media caption for maximum engagement on [platform]. 
Original caption: [paste caption]

Requirements:
- Add relevant emojis (3-5 max)
- Include 5-7 relevant hashtags
- Add a clear call-to-action
- Improve readability with line breaks
- Make it more engaging and conversational
- Keep brand voice: friendly, professional, helpful
- Target audience: small business owners
- Platform: [Instagram/Facebook/TikTok/LinkedIn/Twitter]
```

## Best Practices Reminder:

1. **Hook First:** First line determines if people keep reading
2. **Value Early:** State benefit within first 3 lines
3. **Visual Breaks:** Use line breaks, emojis, bullets for scannability
4. **CTA Clear:** Make action obvious and easy
5. **Authentic:** Sound human, not robotic
6. **Platform-Specific:** Adapt for each platform's culture
7. **Test & Learn:** Try variations and track what works
8. **Engage:** Ask questions, encourage comments
9. **Timing:** Post when audience is active
10. **Consistency:** Maintain brand voice across all platforms
