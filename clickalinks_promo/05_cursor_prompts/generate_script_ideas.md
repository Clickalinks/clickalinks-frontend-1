# Cursor Prompts for Generating Script Ideas

Use these prompts with Cursor AI to generate promotional content for ClickaLinks.

---

## Prompt 1: Generate Video Script Ideas

```
Generate 5 different 30-second video script ideas for ClickaLinks, an affordable UK advertising platform. 
Key points to include:
- £1 per day pricing
- 2,000 advertising spots
- Fair rotation system
- Easy setup process
- Target audience: small businesses, local businesses, online stores
- Tone: professional, energetic, approachable
- Include hook, problem, solution, benefits, and CTA
- Use brand colors: vibrant gradients with orange (#ff6b35), cyan (#06b6d4), blue (#3b82f6), green (#10b981), pink (#ec4899), yellow (#fbbf24)
```

## Prompt 3: Generate Email Campaign Series

```

Email sequence should:
1. Welcome/Introduction email
2. Feature highlight email
3. Social proof/testimonial email
4. Limited-time offer email
5. Final reminder email

Each email should:
- Be 150-250 words
- Include clear subject lines
- Have strong CTAs
- Be mobile-friendly
- Include unsubscribe option
- Use brand colors and professional tone
- Address common objections (cost, effectiveness, setup difficulty)
```
