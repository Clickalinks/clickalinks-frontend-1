FILE NAMING CONVENTIONS:
------------------------

Videos:
- Format: [purpose]_[platform]_[version].mp4
- Example: hero_launch_facebook_v2.mp4

Images:
- Format: [type]_[platform]_[number].jpg/png
- Example: instagram_post_1.jpg

Scripts:
- Format: [type]_[purpose].md
- Example: voiceover_hero_launch.md

Audio:
- Format: [type]_[mood/purpose].mp3
- Example: background_track_energetic.mp3

WHAT TO ADD NEXT:
-----------------

1. SCREEN RECORDINGS (02_raw_media/screen_recordings/):
   - Use OBS, Loom, or screen recording software
   - Record in 1080p minimum
   - Show key features: grid selection, purchase flow, shuffle
   - Keep recordings clean (close unnecessary tabs)

2. LOGOS (02_raw_media/branding/):
   - Export logo in PNG format
   - Primary: colored version on transparent background
   - White: white version for dark backgrounds
   - High resolution (at least 1000x1000px)

3. AUDIO (02_raw_media/audio/):
   - Find royalty-free music (Epidemic Sound, Artlist, YouTube Audio Library)
   - Energetic: upbeat, modern, corporate-friendly
   - Corporate: professional, subtle, background-friendly
   - Save as MP3, 320kbps quality

4. CANVA DESIGNS (03_work_in_progress/canva_designs/):
   - Create designs in Canva
   - Save links or export designs
   - Note: Canva projects are cloud-based, save links here

5. FINAL EXPORTS (04_final_exports/):
   - Export videos in 1080p (1920x1080)
   - Social static images: Instagram (1080x1080), Facebook (1200x628)
   - Audio: MP3, 320kbps, mono or stereo as needed

TIPS:
-----

- Keep original files in 02_raw_media/ (never delete)
- Version control: Add _v2, _v3 to filenames for iterations
- Organize by campaign: Create subfolders for specific campaigns
- Backup: Keep cloud backup of all files
- Naming: Be descriptive, use dates if needed (YYYY-MM-DD)
- Quality: Always export finals in highest quality needed
