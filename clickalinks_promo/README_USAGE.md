## üé¨ Creating Videos

### Step 1: Choose Your Script
1. Open `01_scripts/` folder
2. Pick a script that matches your needs:
   - **Hero Launch** ‚Üí Main promotional video
   - **30sec Benefits** ‚Üí Feature-focused video
   - **Testimonial Style** ‚Üí Customer story video
   - **Quick Tips** ‚Üí Educational short videos
   - **15sec Hook** ‚Üí Social media videos

### Step 2: Record Voiceover
1. Read the script naturally
2. Use recording software (Audacity, phone voice recorder)
3. Save as: `voiceover_[name].mp3`
4. Store in: `02_raw_media/audio/` or `04_final_exports/audio_exports/`

### Step 3: Record Screen (If Needed)
1. Use OBS Studio, Loom, or built-in screen recorder
2. Record: Homepage, purchase flow, grid selection
3. Save as: `homepage_tour.mp4`, `purchase_flow.mp4`
4. Store in: `02_raw_media/screen_recordings/`

### Step 4: Edit Video
1. Use CapCut, Adobe Premiere, or your preferred editor
2. Follow prompts in `video_generation_prompts.md`
3. Import: Voiceover + Screen recording + Background music
4. Add text overlays with brand colors
5. Export as: `[name]_[platform]_1080p.mp4`
6. Store in: `04_final_exports/videos_1080p/`

## üéØ Quick Workflows

### Workflow 1: Create One Video
1. Read `voiceover_hero_launch.md`
2. Record voiceover (5 minutes)
3. Record screen (10 minutes)
4. Edit in CapCut (30 minutes)
5. Export and post

**Total Time:** ~45 minutes

### Workflow 2: Create Social Media Post
1. Generate image using `image_generation_prompts.md` (5 minutes)
2. Pick caption from `social_media_captions.md` (2 minutes)
3. Optimize caption (optional, 3 minutes)
4. Post to social media (2 minutes)

**Total Time:** ~12 minutes

### Workflow 3: Send Email Campaign
1. Choose email from `email_sequence_nurture.md` (2 minutes)
2. Customize template (5 minutes)
3. Set up in email tool (5 minutes)
4. Schedule or send (2 minutes)

**Total Time:** ~14 minutes
