## üìù STEP-BY-STEP: USING CAPCUT TEXT-TO-SPEECH

### Step 1: Prepare Your Script

1. Open your script from `01_scripts/` folder:
   - `voiceover_hero_launch.md`
   - `tiktok_30sec_script.md`
   - `voiceover_30sec_benefits.md`
   - Or any other script

2. Copy the script text
3. Remove timestamps like `[0:00-0:05]`
4. Keep only the spoken text
5. Break into natural sentences

**Example:**
```
Original script:
[0:00-0:05] Hook
"Ever wonder if your ad gets buried while bigger brands dominate?"

[0:03-0:08] Problem Setup
"That's the problem with most advertising platforms."

Prepared for CapCut:
Ever wonder if your ad gets buried while bigger brands dominate? That's the problem with most advertising platforms. Big budgets get prime spots. Small businesses? Stuck in the corner.
```

### Step 2: Open CapCut Project

**Mobile:**
1. Open CapCut app
2. Create new project or open existing
3. Set aspect ratio (9:16 for TikTok, 16:9 for YouTube)

**Desktop:**
1. Open CapCut PC
2. Create new project
3. Set resolution (1080p)

### Step 3: Add Text-to-Speech

**Mobile (CapCut App):**
1. Tap "Text" at the bottom
2. Type or paste your script text
3. Tap the text box
4. Look for "Text-to-Speech" or "Voice" icon
5. Tap it to convert text to speech

**Desktop (CapCut PC):**
1. Click "Text" in the toolbar
2. Add text box
3. Type or paste your script
4. Select the text
5. Look for "Text-to-Speech" or "Voice" option
6. Click to generate speech

### Step 4: Choose Voice

**Available Voice Options:**
- **Male voices:** Deep, professional, friendly
- **Female voices:** Clear, energetic, warm
- **Accents:** British, American, Australian (if available)
- **Languages:** English, Spanish, etc. (depending on version)

**Tips for Choosing:**
- **Energetic videos:** Choose upbeat, friendly voice
- **Professional videos:** Choose deep, authoritative voice
- **TikTok/Reels:** Choose trendy, youthful voice
- **Test different voices:** Try 2-3 options, pick the best

### Step 5: Adjust Voice Settings

**Speed/Pace:**
- Default: Usually good
- Faster: For quick videos (1.1x-1.2x)
- Slower: For emphasis (0.9x-0.95x)

**Pitch:**
- Default: Usually best
- Higher: More energetic
- Lower: More professional

**Volume:**
- Set to 100% (you'll adjust in timeline)

### Step 6: Generate and Review

1. Tap/Click "Generate" or "Create"
2. Wait for processing (usually 10-30 seconds)
3. Listen to the generated voiceover
4. If not satisfied:
   - Try different voice
   - Adjust speed/pitch
   - Re-generate

### Step 7: Edit Text-to-Speech (If Needed)

**To Make It Sound Natural:**

**Add Pauses:**
- Add commas where you want pauses
- Add periods for longer pauses
- Use ellipsis (...) for dramatic pauses

**Example:**
```
Before: "ClickaLinks changes everything. Just ¬£1 per day. Fair rotation means everyone gets seen."

After: "ClickaLinks changes everything... Just ¬£1 per day. Fair rotation means everyone gets seen."
```

**Emphasize Words:**
- Use ALL CAPS for emphasis (sparingly)
- Add punctuation for natural flow

**Break Long Sentences:**
- Split into shorter sentences
- Easier for text-to-speech to handle
- Sounds more natural

### Step 8: Position in Timeline

1. The text-to-speech audio is automatically added
2. Drag to position where you want it
3. Trim if needed (cut beginning/end)
4. Adjust volume if necessary

### Step 9: Sync with Video

1. Add your screen recordings/video clips
2. Match video to voiceover timing
3. Use voiceover as guide for video cuts
4. Adjust video speed to match voiceover pace

## üéØ TIPS FOR BEST RESULTS

### Tip 1: Write Script for Text-to-Speech

**Write Naturally:**
- Use conversational language
- Short sentences work best
- Avoid complex words (if possible)
- Add natural pauses with punctuation

**Example:**
```
Good: "ClickaLinks is affordable. Just ¬£1 per day. No contracts. No hidden fees."

Bad: "ClickaLinks, which is an affordable advertising platform, costs just ¬£1 per day, and there are no contracts or hidden fees."
```

### Tip 2: Add Punctuation for Natural Flow

**Use Punctuation:**
- **Comma (,):** Short pause
- **Period (.):** Medium pause
- **Ellipsis (...):** Longer pause
- **Exclamation (!):** Emphasis, energy
- **Question mark (?):** Natural question tone

**Example:**
```
"Advertising costs too much? We get it. That's why ClickaLinks starts at just ¬£1 per day. No contracts. No hidden fees. Just affordable advertising that works."
```

### Tip 3: Break Script into Sections

**For Long Scripts:**
- Break into 2-3 text boxes
- Generate each separately
- Combine in timeline
- Easier to edit and adjust

**Example:**
```
Text Box 1: "Ever wonder if your ad gets buried while bigger brands dominate? That's the problem with most advertising platforms."

Text Box 2: "Not with ClickaLinks. Watch this. Every 2 hours, our system automatically shuffles ALL ads."

Text Box 3: "Fair advertising starts at ¬£1 per day. Link in bio!"
```

### Tip 4: Test Different Voices

**Try Multiple Options:**
- Generate with 2-3 different voices
- Listen to each
- Pick the one that matches your video tone
- Save your favorite for future use

### Tip 5: Adjust Timing

**After Generation:**
- Listen to the voiceover
- Note the timing
- Adjust video clips to match
- Add pauses if needed (split audio)

## üîß TROUBLESHOOTING

### Problem: Voice Sounds Robotic

**Solutions:**
- Try different voice option
- Add more punctuation for pauses
- Break into shorter sentences
- Adjust speed slightly (0.95x-1.05x)
- Use premium voices (if available in your CapCut version)

### Problem: Voice Too Fast/Slow

**Solutions:**
- Adjust speed in voice settings
- Add more punctuation (slows down)
- Remove some punctuation (speeds up)
- Split into multiple text boxes with different speeds

### Problem: Mispronunciations

**Solutions:**
- Spell out numbers: "one" instead of "1", "pound" instead of "¬£"
- Write out abbreviations: "Click a Links" instead of "ClickaLinks" (if needed)
- Use phonetic spelling for difficult words
- Break into smaller text boxes for problematic sections

### Problem: Voice Doesn't Match Video Tone

**Solutions:**
- Try different voice gender/type
- Adjust pitch (higher = more energetic, lower = more professional)
- Match voice to your target audience
- Test 2-3 options before finalizing
